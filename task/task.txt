В рамках задания требуется ознакомиться с инструментом Ghidra: 
https://github.com/NationalSecurityAgency/ghidra/releases/download/Ghidra_11.0.2_build/ghidra_11.0.2_PUBLIC_20240326.zip

1) Требуется создать проект в Ghidra и загрузить туда бинарный файл binary.
2) Найти исходный код на Гитхаб, опираясь на информацию о бинарном файле 
(например, смотря на код в Ghidra). 
Подсказка: Можно так же посмотреть файлы function_metrics.json и binary_metrics.json (в них находятся 
собранные для бинарного файла метрики).
3) Назвать метрики, которые использовались при поиске исходного кода для бинарного файла.
